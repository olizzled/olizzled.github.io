Explicación de las etiquetas

nav

El nav se compone de dos uno que te dirige directo al indice del sitio
y el otro que te dirige a la explicación de las etiquetas usadas en la pagina de la practica 5.

p

La etiqueta de parrafo se uso en el texto antes de la tabla de los colores en ingles
donde dice "Con estos videos y audios podras aprender la correcta pronunciacion de los siguientes colores en ingles:"

a

La etiqueta a es para enlaces y se utilizo para dirgir del cuadro de navegación que dice index
a la pagina en cuestion

h1

La etiqueta h1 que se usa para encabezados grandes como el titulo de un periodico, etc. Se utilizo
para el titulo de la Practica 5 asi como tambien para el texto que dice "Index" y "Explicación."

table

eran frecuentemente utilizados para la disposición (posicionamiento) de una página. Este uso ha sido desalentado desde HTML 4, y el elemento no debe ser usado para dichos fines.El Elemento de Tabla HTML (<,table>) representa datos en dos o mas dimensiones.

thead

El elemento HTML thead define una sección de encabezado en una tabla . Cuando está presente, debe contener una o más filas que se supone, proveen información de encabezado para las celdas de datos. existen más beneficios que la semántica al realizar el marcado de encabezados y pies en una tabla

tr

El elemento HTML fila de tabla (table row) tr define una fila de celdas en una tabla. Estas pueden ser una mezcla de elementos td y th. Este atributo enumerado especifica como será gestionado el alineamiento horizontal del contenido de cada celda. Hay varios valores.

th

El elemento HTML th define una celda como encabezado de un grupo de celdas en una tabla. La naturaleza exacta de este grupo está definida por los atributos scope y headers.

tbody

El elemento tbody contiene a un bloque de filas (tr) que representaa a la sección del cuerpo de una tabla (table). Este elemento puede ser útil para especificar qué filas proveen información de contenido en la tabla, en contraste con aquellas que son parte del encabezado (header) o el pie (footer).

id

El atributo global id define un identificador único (ID) el cual no debe repetirse en todo el documento. Su propósito es identificar el elemento al vincularlo (usando un identificador de fragmento), en scripts u hojas de estilo (con CSS)
